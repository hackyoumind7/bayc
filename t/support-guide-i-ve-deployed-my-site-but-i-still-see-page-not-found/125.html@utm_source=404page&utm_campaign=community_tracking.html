Web servers and web browsers communicate using a protocol called HTTP - which stands for ‚ÄúHyperText Transfer Protocol‚Äù. Both web browsers and web servers use a feature called headers as part of this protocol. The headers that web browsers send are called ‚Äúrequest headers‚Äù because the browser in making an ‚ÄúHTTP request‚Äù. The web servers headers are called ‚Äúresponse headers‚Äù because web servers send an ‚ÄúHTTP res‚Ä¶

  • the time of the request
  • the timezone the time is in
